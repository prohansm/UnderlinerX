# 工具

## UnderlinerX

### 功能

- 将文件名中的空格替换为下划线

### 使用方法

- Windows

  方法一（推荐）：将需要重命名的文件拖动到 UnderlinerX.exe 文件上

  方法二：双击打开UnderlinerX.exe，跟随提示输入文件名，复制程序的结果

- MacOS

  方法一：使用Terminal执行 UnderlinerX.sh，将需要重命名文件的路径输入到命令行参数

  方法二：双击打开UnderlinerX.sh，跟随提示输入文件名，复制程序的结果
